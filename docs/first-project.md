# 🎯 你的第一个项目：3 分钟让 Agent 帮你做一个网页

> 不需要懂代码。你只说话，Agent 帮你做。3 分钟后你会在浏览器里看到你自己的作品。

---

## 前提

你已经有 Agent 了。如果你还不知道 Agent 是什么，先看 👉 [Agent 是什么？](./what-is-agent.md)

---

## 开始

### 第 1 步：打开你的 Agent

打开你下载的 Agent 软件（不管是 Hermes、Claude Code、OpenCLAW 还是 Codex），确保它正在运行。

> 🤔 不确定自己是什么 Agent？直接对它说「你是什么 Agent？」它会告诉你。

---

### 第 2 步：对 Agent 说第一句话

复制下面这句话，发给你的 Agent：

> **「帮我在桌面上创建一个叫 my-first-page 的文件夹，里面放一个 index.html 文件，内容是：Hello World 用大标题，下面一行小字写『这是我用 Agent 做的第一个网页』，背景用浅蓝色。」**

Agent 会开始干活。你会看到它回复一些东西——不用管它说什么，等着就行。

---

### 第 3 步：让 Agent 打开给你看

Agent 回复「做好了」之类的话后，对它说：

> **「帮我在浏览器里打开这个文件」**

Agent 会自动打开你的浏览器，你会看到：

```
┌─────────────────────────────────────────┐
│                                         │
│         Hello World                      │
│   这是我用 Agent 做的第一个网页            │
│                                         │
│              （浅蓝色背景）               │
│                                         │
└─────────────────────────────────────────┘
```

🎉 **恭喜！你刚刚用 Agent 创建了人生中第一个网页！**

---

## 接着玩：改一改

现在试试让 Agent 帮你改东西。对它说：

> **「把标题颜色改成红色，背景改成白色，在下面加一张图片（用你喜欢的随便一张）」**

Agent 会帮你改好，刷新浏览器你就能看到效果。

---

## 再玩一个：加一个按钮

对它说：

> **「在标题下面加一个按钮，写着『点我』，点击后弹出一个提示框说『你好呀！』」**

刷新看看——点按钮真的会弹窗！

---

## 你会发现什么？

回顾一下，你刚才：

- ✅ 创建了一个文件夹
- ✅ 创建了一个网页文件
- ✅ 改了颜色
- ✅ 加了图片
- ✅ 加了一个能点的按钮

**一行代码都没写。** 全都是你说话，Agent 干活。

---

## 这跟「真正做项目」有什么区别？

**没有区别。** 上面的操作，放大 100 倍就是做一个完整的公司官网：

你：「帮我在桌面创建一个公司官网，有首页、产品页、联系页」
Agent：帮你创建 3 个页面，写好所有代码，在浏览器里打开给你看
你：「产品页的排版不好看，改成两列」
Agent：改好

**原理一模一样：你说 → Agent 做 → 你看 → 不满意 → 再说 → Agent 再改。**
