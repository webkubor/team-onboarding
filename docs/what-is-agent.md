# 🤖 Agent 是什么？跟我用的豆包有什么区别？

> 这是小白看板最重要的一篇。花 3 分钟读完，你就知道这玩意儿到底能干嘛。

---

## 一句话解释

**豆包是聊天机器人，Agent 是能帮你干活的人。**

---

## 用类比来理解

### 豆包 / ChatGPT = 电话里的客服

你问「深圳今天天气怎么样」→ 它告诉你。

你问「帮我写一段文案」→ 它写出来，复制粘贴给你。

**但到此为止了。** 它只能「说」答案，不能「做」事情。

---

### Agent = 请了一个实习生

你对 Agent 说「帮我把这张图片的背景换成白色」→ 它真的打开电脑里的软件帮你处理，存到文件夹里，然后告诉你「好了，在桌面上的『处理好的图片』文件夹里」。

你对 Agent 说「帮我做一个公司官网」→ 它真的帮你创建文件、写代码、在浏览器里打开让你看到效果。

**Agent 不只是给你答案，它帮你把事办了。**

---

## 一张表看懂区别

| | 豆包 / ChatGPT | Agent |
|---|---|---|
| 你能对它做什么 | 聊天、问问题 | 聊天、问问题、**让它操作电脑** |
| 它给你的结果 | 一段文字 | 一个**已经完成的东西**（文件、网页、处理好的图片） |
| 它能动你的电脑吗 | ❌ 不能 | ✅ 能（读文件、写文件、改代码） |
| 需要你复制粘贴吗 | 需要，它说的你得自己手动做 | 不需要，它直接帮你做了 |

---

## 举个真实的例子

### 场景：你想做一个简单的个人主页

**用豆包：**

你：「帮我写一个个人主页的代码」
豆包：「好的，这是 HTML 代码：`<html>...`」
你：（一脸懵）「这串东西怎么用？我往哪贴？」

**用 Agent：**

你：「帮我在桌面创建一个个人主页，包含我的名字和一张头像」
Agent：「好的。我先在桌面创建一个 index.html 文件……做好了。我在浏览器里打开给你看看？」
你：「好」
Agent 帮你打开浏览器 → 你看到一个漂亮的个人主页 🎉
你：「把标题改成红色，加粗」
Agent：「好的」→ 一秒改好，刷新给你看

**区别就是：豆包给你配方，Agent 帮你把菜炒好端上来。**

---

## 那 Agent 是怎么「干活」的？

Agent 运行在你的电脑上（或者别人的服务器上），它能：

- 📂 读你电脑上的文件
- ✏️ 写文件、改文件
- 🌐 帮你访问网页
- 💻 运行命令（就像你自己在键盘上敲命令一样）

听起来很可怕？**但其实你跟它说的每一句话，它才敢动。你不说，它不动。**

就像你请了个实习生——你不交代任务，他就坐着发呆。

---

## 我需要懂代码才能用 Agent 吗？

**不需要。**

你只需要**说清楚你要什么**，Agent 帮你搞定代码部分。

就像你不需要懂发动机原理才能开车——你只需要知道「踩油门就走，打方向盘就转」。
